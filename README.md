# django_insta
An Instagram-like Web Application with Dango and React.
  
## Feature List

- 유저들의 포스팅 목록을 제공하는 API로 메인 화면에 게시물 표시하기
- 회원가입 API로 유효성 검사 및 새로운 회원 정보 등록하기 

## Tech Stack   

- Django 3.0.14
- Django Rest Framework
- React
- HTML, CSS
- Doker
